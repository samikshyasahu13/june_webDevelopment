# StormSense - Smart Umbrella with Weather Alerts

## Developer
Samikshya Sahu

## University
Ravenshaw University

## Course
B.Sc. Computer Science

## Project Description
StormSense is a smart umbrella that alerts users before rainfall through weather notifications. It includes GPS tracking, wind-resistant design, auto-open technology, and mobile connectivity.

## Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design
- Tailwind CSS

## Features
- Weather Alert Handle
- GPS Tracking
- Wind Resistant Design
- Auto Open System
- Mobile App Connectivity
- Long Battery Life
